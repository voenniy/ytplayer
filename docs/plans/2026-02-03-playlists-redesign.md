# Плейлисты: редизайн

**Цель:** Убрать пустой Sidebar, перенести плейлисты в табы основного контента. Добавить полный флоу: создание, добавление треков, запуск, редактирование.

---

## Новый лейаут

### Десктоп (>= 768px)
- Две колонки: Основной контент | Очередь
- Sidebar удаляется
- В основном контенте сверху — два таба: **Поиск** | **Плейлисты**
- Таб "Поиск" — строка поиска + результаты
- Таб "Плейлисты" — список плейлистов / содержимое плейлиста

### Мобильный (< 768px)
- Нижний таббар: Поиск | Плейлисты | Очередь (без изменений)
- Вкладка "Плейлисты" рендерит тот же компонент плейлистов

---

## UX-флоу

### Создание плейлиста
- Поле "Новый плейлист" + кнопка "+" вверху вкладки Плейлисты

### Добавление трека в плейлист
- В TrackList кнопка ListPlus заменяется на DropdownMenu
- Пункты: «В очередь» | «В плейлист →» (подменю: список плейлистов + «Создать новый»)
- На мобильных — стандартный shadcn dropdown (адаптивный)

### Запуск плейлиста
- Клик по плейлисту → треки заменяют очередь, первый трек играет

### Просмотр/редактирование содержимого
- Кнопка «Открыть» у плейлиста → экран с треками
- Каждый трек: drag-handle (≡) для порядка, обложка, название, кнопка удаления (×)
- Кнопка «Воспроизвести всё» вверху
- Кнопка «Назад» → возврат к списку плейлистов
